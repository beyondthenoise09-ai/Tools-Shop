## How to Use

1. **Open** ToolsShop in your browser
2. **Search** for a tool using the search bar
3. **Upload** your file (images or PDFs)
4. **Process** with customizable options
5. **Download** your processed file

All files are automatically deleted after processing - they never leave your device!
